What is ResTool NXT

The newest in a long line of helper programs for NIU's Residential Technology Helpdesk, ResTool NXT is an automated program for running anti-malware and troubleshooting processes on Windows computers. It contains automation routines for several popular anti-malware programs, many system tools and several other miscellaneous programs or processes, all wrapped up in a convenient and easy-to-understand format.
